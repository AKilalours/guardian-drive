# Guardian Drive -- Live Demo

**Built by Akilan Manivannan & Akila Lourdes Miriyala Francis**
Long Island University, Brooklyn, NY

A multimodal AI research prototype for driver physiological monitoring.

- Task B: WESAD TCN low-arousal classifier (AUC 0.9738, window-level split)
- Waypoint predictor: causal transformer on nuScenes mini (ADE 7.70m)
- LibTorch/MPS benchmark results
- Full system documentation

**Not a medical device. Not clinically validated.**

GitHub: https://github.com/AkilanManivannanak/guardian-drive
